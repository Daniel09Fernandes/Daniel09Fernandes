### 🦖 Sobre Mim
Desenvolvedor de Minas Gerais com sólida experiência em arquitetura de software. Como **Embarcadero MVP**, dedico parte do meu tempo contribuindo para o ecossistema Delphi, mas também domino stacks modernas para soluções Fullstack e integrações complexas.

* 🚀 **Foco Principal:** Delphi (VCL/FMX) e Backend.
* 🛠 **Linguagens de Domínio:** C#, Python, PHP e Javascript.
* 🌱 **Evoluindo em:** Angular e .NET Core.
* 💞️ **Open Source:** Mantenedor do [Componente DinosOffice](https://github.com/Daniel09Fernandes/ComponentDinosOffice-OpenOffice).
